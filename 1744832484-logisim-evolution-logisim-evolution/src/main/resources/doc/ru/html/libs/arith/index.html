Библиотека Арифметика

Библиотека Арифметика включает комбинационные компоненты, которые выполняют арифметические действия над значениями в виде беззнаковых чисел и в дополнительном коде.

  Вычитатель
  Множитель
  Делитель
  Отрицатель
  Компаратор
  Сдвигатель
  Сумматор битов
  Искатель битов
