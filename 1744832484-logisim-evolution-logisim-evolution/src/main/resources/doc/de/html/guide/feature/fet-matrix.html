Komponenten in einer Matrix platzieren

In Logisim-evolution ist es möglich, Matrixbauteile auf Ihrem Schaltplan zu platzieren. Gehen Sie dazu wie folgt vor:
Wählen Sie das gewünschte Werkzeug in der Navigationsleiste aus, z.B. die LED, und Bewegen Sie es auf die Arbeitsfläche und drücken Sie vor dem Klicken die X-Taste, um es fallen zu lassen. Der Schatten des Bauteils verwandelt sich in eine kleine 2 mal 2 Matrix des Bauteils.

Klicken Sie oben links auf die Position, an der Sie die zukünftige Matrix platzieren möchten. Und füllen Sie die Daten des Formulars aus. Zum Beispiel: 4x5 Elemente mit einem Abstand von 4.

Wenn die Selbstbeschriftung aktiviert ist, werden die Komponenten mit _X_Y, Beispiel beschriftet: Led_X1_Y0
