Speicherinhalte bearbeiten

Sie können Speicherinhalte mit dem Schaltwerkzeug bearbeiten (), dies ist aber durch den vorhandenen Platz eingeschränkt: abgesehen von den einfachsten Änderungen, werden Sie wohl auf den eingebauten Hex-Editor zurückgreifen.

Auswahl der Adresse

Um eine Adresse auszuwählen, klicken Sie außerhalb des Anzeige-Rechtecks oder auf die Adressspalte. Logisim zeichnet ein rotes Rechteck um die obere Adresse.
Um den Editor zu verlassen, klicken Sie einfach an einer anderen Stelle des Schemas.

  • Wenn Sie einen hexadezimalen Wert eingeben, positioniert sich der Editor an der gewünschten Adresse.
  • Mit der Eingabetaste oder Strg+Abwärts-Pfeil können Sie eine Adresse nach unten scrollen.
  • Mit der Rücktaste- oder Strg+Aufwärts-Pfeil können Sie eine Adresse nach oben scrollen.
  • Das Mausrad ermöglicht das gleiche Scrolling.
  • Mit der Leertaste wird eine Seite nach unten geblättert.

Ändern von Daten

Um die Daten zu ändern, klicken Sie auf eines der Datenfelder. Logisim zeichnet ein rotes Rechteck um die Box.

Sie können dann den Wert der Speicherzelle in hexadezimaler Form eingeben.

  • Mit der Eingabetaste oder dem Strg+Abwärts-Pfeil scrollt den Cursor eine Zeile nach unten.
  • Mit der Rücktaste- oder Strg+Links-Pfeil blättert den Cursor ein Feld zurück.
  • Mit der Leertaste oder Strg+Rechts-Pfeil scrollt den Cursor eine Zelle vorwärts.
  • Das Mausrad oder Strg+Aufwärts-Pfeil , Strg-Abwärts-Pfeil nach unten erlaubt es, eine Zeile nach oben/unten zu bewegen.

Um den Editor zu verlassen, klicken Sie einfach an einer anderen Stelle des Schemas.
