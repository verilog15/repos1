Schritt 5: Der Einzelschrittmodus

Unter bestimmten Studienbedingungen oder bei subtilen Problemen im Zusammenhang mit der Ausbreitung der Signale ist es sehr nützlich, diese Phänomene beobachten zu können. Logisim simuliert auf eine leicht naive Weise die Ausbreitung in Gates. (die Gatter haben alle die gleiche Ausbreitungsverzögerung), aber es erlaubt trotzdem, die Hauptprobleme in Bezug auf diese Parameter zu beobachten.

Für das Beispiel werde ich Ihnen nur die für diese Übung notwendigen Tastenkombinationen geben; beachten Sie, dass es für jede Tastenkombination ein gleichwertiges Untermenü im Menü gibt | Simulieren |.

Wenn Sie die obige Schaltung aufbauen, werden Sie feststellen, dass jede Änderung von 0 auf 1 des Eingangs e das Flip-Flop D stört.

Um das Phänomen zu beobachten, gehen Sie wie folgt vor: Bringen Sie den Schaltkreis in den Zustand, in dem e gleich 0 ist. Führen Sie dann Strg -E aus, um die automatische Simulation zu deaktivieren. Drücken Sie den Eingang e, um ihn auf 1 zu schalten. Es passiert nichts, das ist normal. Dann die Kombination Strg-I verwenden und beobachten. Machen Sie das so oft wie nötig.

Sie werden blaue Kreise an einigen Fadenenden bemerken, diese zeigen an, dass sich der Zustand dieses Fadens in diesem Schritt der Simulation ändert. Sie können den kurzen Durchgang durch 1 des Ausgangs des UND-Gatters beobachten.

Mit dieser Methode ist es auch möglich, die asynchrone Zähleroperation oder die Ausbreitung von Holdback in einem Addierer zu beobachten.

Nun, da Sie mit dem Tutorial fertig sind, können Sie mit Logisim experimentieren, indem Sie Ihre eigenen Schaltungen bauen. Wenn Sie Schaltungen mit anspruchsvolleren Funktionen bauen wollen, dann sollten Sie durch das restliche Hilfesystem navigieren, um zu sehen, was Sie sonst noch tun können. Logisim ist ein leistungsfähiges Programm, mit dem Sie große Schaltungen aufbauen und testen können; dieser Schritt-für-Schritt-Lernprozess kratzt nur an der Oberfläche.
