Panneau du fichier mémoire

Cette fenêtre s'ouvre quand la reconnaissance de fichier n'arrive pas à déterminer le type de la structure des données. Elle se compose d'une serine de case à cocher pour préciser les options de conversion. De deux onglets d'affichage hexadécimal qui présente les données du fichier et d'une zone de texte qui affiche les erreurs éventuellement détectées.

Les quatre boutons radio permettent de sélectionner un des types de structure de fichier présentée précédemment. Pour chaque type des cases à cocher s'activeront. Elles permettent de préciser les options spécifiques.
Vous trouvez les informations dans les pages dédiées à ces types

  • v2.0 raw
  • v3.0 hex words plain/adressed
  • v3.0 hex bytes big-endian/little-endian
  • Binary data big-endian/little-endian
  • Ascii byte escape big-endian/little-endian

Le panneau d'affichage hexadécimal vous permet d'observer de quelle manière Logisim décode le fichier.

Et de le comparer à sa version originale.
