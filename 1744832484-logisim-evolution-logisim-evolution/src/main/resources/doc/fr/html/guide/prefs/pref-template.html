L'onglet Modèle

Un modèle est un fichier Logisim utilisé comme document de base à chaque fois que Logisim crée un nouveau projet. De plus, si vous disposez d'un fichier Logisim existant avec une configuration de l'environnement étrange , vous pouvez "réinitialiser" à l'aide du bouton  Réinitialiser selon modèle  de la fenêtre options de projet | Projet |→| Options ...  |→| Réinitialiser  |.

Bien que les modèles soient également utiles dans d'autres situations, ils sont particulièrement adaptés à une utilisation en classe, où un instructeur peut souhaiter distribuer un modèle de départ pour les élèves. Cela est particulièrement probable si la classe utilise fortement Logisim, y compris de nombreuses fonctionnalités plus avancées, auquel cas la configuration par défaut peut s'avérer trop simple. Les modèles peuvent également être utiles dans la classe lorsque l'instructeur ouvre un fichier soumis par un étudiant qui a reconfiguré l'environnement de manière significative.

Par défaut, l'option Modèle par défaut complet sera sélectionnée. Ce modèle est fourni avec Logisim. Si vous voulez une configuration simple, vous pouvez choisir l'option Modèle vide. Mais si vous souhaitez désigner un autre fichier à utiliser comme modèle, sélectionnez le fichier via le bouton  Sélectionner ... , puis choisissez l'option Modèle spécifique.

Il est aussi possible de sélectionner un modèle par un paramètre en ligne de commande : -template templateFile

Suite: L'onglet International.
