Les barres d'outils

Logisim-evolution dispose de plusieurs barres d'outils. La barre d'outils principale est configurable et met à disposition un accès rapide aux outils les plus utilisés. Ceux-ci restent aussi toujours disponibles dans le panneau de navigation.


La barre dans le mode "édition du dessin"


La barre dans le mode "édition de l'apparence"

Et enfin deux autres barres apparaissent selon l'onglet du panneau de navigation. Elles sont présentées ci-dessous.

  

La barre d'outils principale

Cette barre d'outils à deux présentations (voir ci-dessus) selon le mode d'édition choisit "édition du dessin" ou "édition de l'apparence".

La barre dans le mode "édition du dessin"

Dans le mode "édition du dessin", elle est personnalisable à l'aide du menu | Projet |→| Options | onglet | Barre d'outils | Le processus est en présentée par la page L'onglet Outils. Ci-dessous nous présentons brièvement les outils présents par défaut dans la barre. Il existe beaucoup d'autres composant pour construire des circuits. Vous trouverez beaucoup plus d'information sur l'ensemble des outils dans la Référence de la bibliothèque.

Outil Pousser : Ctrl-1 Cet outil permet d'interagir avec les composants de simulation. Par exemple si vous poussez sur une connexion du circuit celle-ci basculera d'un état à l'autre ou si vous poussez sur un câble vous pourrez connaitre la valeur des signaux transportés. Plus d'information sur le lien pousser de la librairie base
L'outil Sélection : Ctrl-2 Cet outil est particulièrement utile au moment de la conception des circuits, il permet de poser,déplacer ou sélectionner les éléments. Il sert aussi à câbler le circuit. Plus d'information sur le lien sélect de la librairie base.
Outil câblage : Ctrl-3 Cet outil permet de placer des câble sur le schema .Plus d'information sur le lien text de la librairie base
Outil Text : Ctrl-4 Cet outil permet de placer du text sur les dessins et de modifier les labels des composants.Plus d'information sur le lien text de la librairie base
Outil broche : Ctrl-5 / 6 Cet outil permet de placer des broches de connexion sur le schéma. Les connexions sont de deux types : entrée ou sortie. Elles sont présentées dans cette page Broche de la librairie câblage.
Outils Not,And,Or : Ctrl-7 / 8 / 9 / 0 Ces outils permettent de placer les portes logique de base sur le dessin. Les portes logiques sont présentées dans la librairie Portes logiques.
Outils Add D FlipFLop :Ces outils permettent de placer une bascule D sur le dessin. Les bacules logiques sont présentées dans la Librairie memoire / sequenciel
Outils Add Register : Ces outils permettent de placer un registre sur le dessin. Les registres sont présentées dans la Librairie memoire / sequenciel

La barre dans le mode "édition de l'apparence"

Dans ce mode ou nous pouvons éditer le symbole d'un sous circuit. La barre propose des outils de dessins. Vous trouverez un peu plus d'information sur l'usage dans Editer l'apparence d'un sous-circuit

Outil Sélection : Sélectionne, déplace, copie, colle les symboles.
Outil Text : Insert ou édite un text.
Outil Ligne : Crée un segment de ligne.
Outil Courbe : Crée une courbe de Bezier. Le premier clic et glisser spécifie le début de la ligne, le second clic et glisser termine la ligne et définit la courbure. Un clic sur la ligne vous montre les trois points de contrôle. Maj et clic sur point central de contrôle imposent une courbe symétrique. Alt et click sur point central dessine la courbe au travers du point de contrôle (sous la souris).
Outil Polylignes : Crée une séquence de lignes connectées, chaque clic commence une nouvelle ligne. Un double-clic termine la séquence.
Outil rectangle : Crée un rectangle ou un carré (Maj) en glissant depuis un angle vers l'angle opposé.
Outil rectangle : Crée un rectangle ou un carré à bord arrondi (Maj) en glissant depuis un angle vers l'angle opposé.
Outil ovale : Crée un ovale ou un cercle (Maj) en glissant depuis un angle vers l'angle opposé.
Outil polygone : Crée un polygone, chaque clic commence une nouvelle face. Un double-clic termine la séquence.
Composants dynamiques Place un composant dynamique. click ouvre une fenêtre pour la sélection des composants dans les sous-circuits.

La barre d'outils des onglets du panneau de navigation

Cette zone de barre d'outils affiche deux barres selon l'onglet sélectioé du programme. La première a cinque boutons et est affichée quand le programme est dans le mode édition,dessin. Ce sont des raccourcis de certains des sous-menu du menu | Projet |.

Ajouter un circuit Ajoute un nouveau circuit à nos projets.
VHDL: Ajoute un nouveau circuit VHDL à nos projets.
Monte le circuit dans la hiérarchie du projet.
Descends le circuit dans la hiérarchie du projet.
Mode d'édition: Bascule du mode schema au mode édition de l'apparence alternativement.
Supprime un circuit dans la hiérarchie du projet.

La seconde est affichée quand l'onglet simulation est sélectioné. Ce sont des raccourcis de certains des sous-menus du menu | Simulation |

Enclencher/déclencher : Ctrl-E Enclanche ou déclanche la simulation.
Simulation pas-à-pas : Ctrl-I Avancer d'un pas dans la simulation.
Activer/désactiver tic : Ctrl-K Actives ou désactive le fonctionnement des horloges.
Un seul tic : Ctrl-T Avance les horloges d'un tic.
'Clk' un pas : F2 Avance la simulation d'un cycle de l'horloge.

Vous trouverez plus d'information dans la page sur le menu simulation
