函數過程列表

該功能可顯示源碼文件中的所有函數、過程與方法名，並支持匹配查找快速定位，Pascal 源文件還支持類名、屬性和接口等。默認顯示對像為當前源文件。用戶可選擇工程/工程組中的其他單一文件或所有文件、以及所有打開文件等。

另外，該專家將在 IDE 的編輯器中加入一工具欄，實時顯示在編輯的 Pascal 源文件光標處的類型聲明以及函數過程聲明等，可通過下拉的方式快速定位。

  • 跳轉至所選過程：在 IDE 中跳轉到所選擇的過程實現代碼處。

  • 複製函數或過程名：複製所選擇的函數或過程名稱到剪貼板。

  • 按函數/過程名開頭匹配：查找的時候，輸入的字符從函數/過程名的頭部開始匹配，函數/過程名頭部含有輸入字符的才能顯示。

  • 按函數/過程名的所有位置匹配：查找的時候，輸入的字符匹配函數/過程名的任何部分，函數/過程名含有輸入字符的才能顯示。

  • 在編輯器中顯示函數列表工具欄：是否允許在 IDE 的代碼編輯器中顯示函數列表工具欄。此工具欄能實時顯示在編輯的 Pascal/C/C++ 源文件光標處的類名或函數過程名，並能通過下拉的方式快速定位。同時還有按鈕允許快速跳至當前 Pascal 單元的 interface 和 implementation 處。

  • 字體：更改列表框字體。

  • 幫助：顯示幫助信息。

  • 退出：關閉此窗口。

  • 隱藏/顯示預覽窗口：隱藏或顯示選中函數/過程實現部分的預覽欄。當所選擇的過程函數所在的文件未在 IDE 中打開時，預覽不可用。

  • 隱藏/顯示匿名函數：隱藏或顯示列表中的匿名函數。

  • 預覽窗口在底部：將預覽欄位置設定為列表底部。

  • 預覽窗口在右側：將預覽欄位置設定為列表右側。
