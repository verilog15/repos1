Il peut également obtenir les relations entre les packages et les relations entre les packages et ses unités.<BR><BR>Le résultat analysé peut être enregistré dans un fichier ARF. Et nous pouvons créer un fichier SMR à partir d'un fichier ARF pour interroger / afficher les relations entre les fichiers exécutables et les fichiers source ultérieurement.<BR><BR> Pour une situation courante, vous pouvez ouvrir un ou plusieurs fichiers exécutables dans la page "Analyser les fichiers exécutables" et les analyser. Le résultat analysé peut être enregistré dans un fichier ARF et rouvert dans la page "Afficher le résultat analysé".<BR><BR>L'ARF est généré par le résultat analysé, qui ne contient que les relations entre les exécutables et le fichier source Pascal NOM, AUCUN chemin Informations et autres types de fichiers. Pour une situation avancée, vous pouvez ouvrir un fichier ARF dans la page "Modifier les relations source-module" pour le modifier et générer un fichier SMR, qui contient les relations entre les exécutables et les informations de chemin complet des fichiers source. Vous pouvez ensuite afficher les fichiers SMR dans la page "Afficher les relations source-module" ultérieurement.
